## Project Description

This project demonstrates scalable big data analysis using PySpark on the Amazon Sales dataset. The primary objective of this project is to process and analyze large-scale sales data efficiently using distributed computing techniques.

The dataset contains sales-related information such as product categories, order status, sales amount, fulfilment methods, states, cities, and customer order details. PySpark was used to handle large data efficiently and perform scalable computations.

The project includes several stages such as data loading, data cleaning, exploratory data analysis, aggregation, trend analysis, and data visualization. Various business insights were generated including category-wise sales analysis, state-wise sales analysis, quantity analysis, fulfilment analysis, order status distribution, and daily sales trends.

Data visualization techniques such as bar charts, pie charts, line graphs, and area graphs were used to better understand customer purchasing patterns and revenue trends.

This project demonstrates how big data technologies like PySpark can efficiently process large datasets and support business decision-making through scalable analytics.
